STEP 3 — Build synthetic void catalogue from group positions

We have no external void catalogue — build voids directly from the
TNG300 group distribution using an underdensity grid approach:

    box_size = 205.0  # TNG300-1 box size in Mpc/h (comoving)

    a) Divide box into 20×20×20 grid cells
       (cell size ~10 Mpc/h, comparable to small voids)

    b) Count groups per cell using np.histogramdd on 
       [pos_x, pos_y, pos_z] with range [[0,205],[0,205],[0,205]]

    c) Identify void cells: cells where group count = 0
       (genuine underdensities in the simulation)

    d) For each void cell, assign:
         void centre = cell centre coordinates (x, y, z) in Mpc/h
         R_void      = draw from log-normal:
                       mean = log10(25), sigma = 0.20 dex
                       clip to [15, 80] Mpc/h
                       (published TNG300 void size distribution,
                        Cautun et al. 2018)

    e) Keep only void cells that have at least MIN_CLUSTERS_PER_VOID
       groups within the 2–4×R_void annulus (will be checked in 
       build_mock_void_table)

Save void list as: data/tng300_mock_voids.csv
Columns: [void_id, cx_mpc, cy_mpc, cz_mpc, R_void_mpc]

Print: "Mock voids identified: N = {n} underdense cells"
