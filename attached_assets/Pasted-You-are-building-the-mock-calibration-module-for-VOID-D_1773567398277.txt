You are building the mock calibration module for VOID-DOMAIN v1.
This must complete before any real data is downloaded.
All parameters come from void_domain/manifest.py — import and use them.

Goal: establish the null r distribution and expected scatter in slope α

Output: calibrated r thresholds and scatter percentiles for the decision tree.

STEP 2 — download_tng300_data()

Download two files from TNG300-1, snapshot 67 (z ≈ 0.5):

  File 1 — Group/halo catalogue (FoF + Subfind):
    https://www.tng-project.org/data/downloads/TNG300-1/
    Specifically: groups_067 HDF5 catalogue (Group_M_Crit200, GroupPos)
    Save to: data/tng300_groups_snap67.hdf5
    Checkpoint: checkpoints/tng300_groups_downloaded.pkl (flag True when done)

  File 2 — Void catalogue (try in order):
    Primary:   https://catalogs.iate.conicet.unc.edu.ar  (Rodriguez-Medrano 2024)
               Look for TNG300 void catalogue; save to data/tng300_voids.fits or .csv
    Fallback:  If primary unavailable, print "VOID CATALOGUE NOT FOUND — switching
               to synthetic mock" and call _build_synthetic_mock() instead.

  If either download fails (timeout, 404, auth), catch the exception, print the
  URL and error, and proceed to fallback. Do NOT crash.

---

STEP 3 — build_mock_void_table()

Using downloaded (or synthetic fallback) data:

  For each void in the TNG300 catalogue:
    1. Get void centre (RA/Dec or comoving XYZ), R_void (catalogue R_eff).
    2. Find all TNG300 FoF groups within the 2–4 × R_void annulus
       (use ANNULUS_INNER_FACTOR and ANNULUS_OUTER_FACTOR from manifest).
    3. Sum their Group_M_Crit200 masses → M_surrounding.
    4. Exclude voids with fewer than MIN_CLUSTERS_PER_VOID groups in annulus.

  Output: pandas DataFrame with columns:
    [void_id, R_void_hmpc, M_surrounding_1e14Msun, z_void, N_groups_in_annulus]

  Save to: checkpoints/mock_void_table.pkl
  Print: "Mock void table built: N = {n} voids after cuts"

STEP 4 — _build_synthetic_mock() (fallback only)

If the TNG300 void catalogue is unavailable:
  Generate N = 1200 synthetic voids drawn from published TNG300 void statistics:
    - R_void: log-normal, mean=35 h^-1 Mpc, sigma=0.25 dex (Cautun et al. 2018)
    - M_surrounding: log-normal, weakly correlated with R_void (r=0.0, null case)
    - Add Poisson scatter to M_surrounding
  Print: "SYNTHETIC MOCK ACTIVE — real TNG300 data unavailable"
  Save void table as above.

---

STEP 5 — run_mock_fit()

On the mock void table:

  Part A — Real mock fit:
    Fit log10(R_void) = α·log10(M_surrounding) + β using scipy ODR.
    Record: α, σ_α, Pearson r, reduced χ²/dof.

  Part B — Null distribution (1000× shuffle):
    Shuffle M_surrounding labels 1000 times (RANDOM_SEED from manifest).
    For each shuffle: fit ODR, record α_shuffle and r_shuffle.
    Store all 1000 r_shuffle values in array null_r_dist.

  Part C — Compute:
    null_r_median   = np.median(null_r_dist)
    null_r_95pct    = np.percentile(null_r_dist, 95)
    null_r_99pct    = np.percentile(null_r_dist, 99)
    mock_real_r     = Pearson r from real mock fit

  Save: checkpoints/mock_fit_results.pkl (all of the above)

STEP 6 — finalise_thresholds()

Using the mock null distribution, compute calibrated thresholds:

    FINAL_R_OUTCOME_B_LO = null_r_95pct

      Outcome C (FAIL):    r ≤ {FINAL_R_OUTCOME_B_LO:.4f}
    ================================================

  IMPORTANT: Do NOT write these values back to manifest.py yet.
  Print them only. Graham reviews before any threshold is locked.
